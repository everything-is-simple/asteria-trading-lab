## 分月审计

### 1976-01

| 检查项 | 结果 |
|---|---|
| `source_image_checked` | 通过。`B0278.jpg` 与月报一致。 |
| `action_chain_segmented` | `—1 -> —3 -> —5 -> 0`。 |
| `method_candidate_claim` | 可讨论 `trend_probe_entry / trend_confirmation_add / exit_on_rhythm_failure`。 |
| `pm_required_decision` | true。虽为轻仓，但加空与清零仍需要 PM 解释库存链。 |
| `boundary_warning_present` | `do_not_infer_structure_from_short_probe_only`。 |

裁决：升级为 `conditional`。它具备轻仓试探、连续加空、月底清零的 Method / PM 样本价值，但结构背景较弱，不能视为干净趋势样本。

### 1976-02

| 检查项 | 结果 |
|---|---|
| `source_image_checked` | 通过。`B0289.jpg` 与月报一致。 |
| `action_chain_segmented` | `—2 -> —5 -> —10 -> —12 -> —7`。 |
| `method_candidate_claim` | 可讨论 `trend_probe_entry / trend_confirmation_add / distribution_reduce`。 |
| `pm_required_decision` | true。空头库存从轻仓推进到 `—12` 后部分回补，必须区分中心单、加码单和减仓。 |
| `boundary_warning_present` | `do_not_write_center_position_into_malf`。 |

裁决：升级为 `conditional`。它比 1 月更有空头库存节奏意义，也更需要 PM；但没有真实 MALF 快照前，不能升为 `suitable`。

## 当前结论

- `1976-01/02` 已具备“值得进入 Method / PM 讨论”的最低证据，因此不再保留为 `sampled_unknown`。
- `1976-01` 是弱条件样本：重点在轻仓试探和清零边界，不能当作趋势正样本。
- `1976-02` 是较强条件样本：空头库存推进清楚，但中心单和加码单解释必须留在 PM。
- 当前剩余 `sampled_unknown` 只保留 `1976-09`；下一步应先处理其交易单位变更、除权和少量成交事实，再决定是否升级。

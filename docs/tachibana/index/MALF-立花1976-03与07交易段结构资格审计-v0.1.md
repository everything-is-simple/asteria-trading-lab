# MALF-立花 1976-03 与 07 交易段结构资格审计 v0.1

## 版本定位

- 本文件是 [MALF-立花结构资格样本表 v0.1](./MALF-立花结构资格样本表-v0.1.md) 的交易段级补充审计。
- 它专门处理两类 reset 样本：`1976-03` 的清零后新库存种子，以及 `1976-07` 的清零、重建、加码、减仓链。
- 本文件不修改 MALF 主定义，不新增交易规则，不输出买卖信号，不写 A 股制度适配。

## 为什么把 1976-03 与 1976-07 放在一起

这两个样本共同回答一个边界问题：清仓之后，旧节奏是否还可以延续到新试探？

前置过滤器必须给出保守答案：

1. 清零是 PM 状态机事件，应触发 `reset_after_clear`。
2. 清零后的新仓位可以作为 `inventory_seed / open_center_candidate`，但不能自动并入旧中心单。
3. MALF 可以描述清零前后价格结构是否仍有资格进入讨论，但不决定“重新进场是否正确”。

因此，本文件把 `1976-03` 和 `1976-07` 都拆成段级样本，校准“旧段终止”和“新段资格”之间的闸门。

## 当前结论

- `1976-03/07` 证明：清零之后，旧段必须结束，新试探必须重新取得结构资格。
- 六段均为 `conditional`，没有一段应升为 `suitable`。
- MALF 可以辅助判断清零前后是否有结构背景，但 `reset_after_clear`、`inventory_seed`、中心单和加码单归 Method / PM。
- 前置过滤器必须防止两种误读：把清零当成 MALF 方向判断，把清零后的新仓位并回旧中心单。

# MALF-立花 1976-11 交易段结构资格审计 v0.1

## 版本定位

- 本文件是 [MALF-立花结构资格样本表 v0.1](./MALF-立花结构资格样本表-v0.1.md) 的交易段级补充审计。
- 它不再只问“1976-11 这个月是否适合立花法”，而是拆成多个交易段，分别判断其 `tachibana_applicability`。
- 本文件不修改 MALF 主定义，不新增交易规则，不输出买卖信号，不写 A 股制度适配。

## 为什么要拆段

`1976-11` 不是单一结构样本。月内至少发生了：

1. 承接 10 月 `—24` 的右侧仓位，并快速扩大到 `—200`。
2. `—200 -> 0` 的一次性清仓。
3. 清仓后 `—5 -> 0 -> —5` 的小仓重试。
4. `—5 -> 35 —` 的方向切换和新仓位骨架形成。

如果整月只给一个 `conditional`，会掩盖每个交易段的结构资格差异。前置过滤器需要能回答：同一个月里，哪一段值得 Method / PM 讨论，哪一段只能作为 PM 风险样本，哪一段代表旧节奏结束和新节奏开始。

### 1976-11-A：极端同向加码段

| 检查项 | 结果 |

| `action_chain_segmented` | `—24 -> —26 -> —28 -> —48 -> —98 -> —200`。 |
| `method_candidate_claim` | `trend_confirmation_add`。 |

## 当前结论

- `1976-11` 证明：前置过滤器必须支持交易段级资格判定。
- 同一个月内可以同时存在 `alive_wave`、`stagnation / break_candidate`、`transition / unknown`、`break_birth`。
- 四段均为 `conditional`，没有一段应升为 `suitable`，因为每段都强依赖 PM。
- `1976-12`、`1975-06`、`1976-04/05` 与 `1976-03/07` 已按同样方法完成第一版段级拆分；横向收束见 [MALF-立花结构资格横向判读矩阵 v0.1](./MALF-立花结构资格横向判读矩阵-v0.1.md)。

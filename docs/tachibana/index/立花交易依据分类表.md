## 分类使用原则

- 同一笔交易可以同时带有一个动作分类和一个仓位管理解释，例如 `trend_confirmation_add` + 中心单/加码单。
- 月表只能证明动作事实；原文锚点负责证明这种动作在立花方法里有位置。
- MALF 只描述结构位置，不能直接承接心理、资金管理、锁单、中心单。
